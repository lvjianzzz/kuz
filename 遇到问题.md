+ vue 中 通过索引值给数组赋值时的问题
出现问题的写法
    ```
             created() {
                for (var i = 0; i < 125; i++) {
                    this.list[i] = {
                        id: Random.guid(),
                        line: Math.floor(i / 25) + 1,
                        row: Math.floor(i % 25),
                        isselect: false
                    }
                }
            },
    ```
修改后的写法
    ```
       created() {
                for (var i = 0; i < 125; i++) {
                    this.list.push({
                        id: Random.guid(),
                        line: Math.floor(i / 25) + 1,
                        row: Math.floor(i % 25),
                        isselect: false
                    })
                }
            },
    ```
